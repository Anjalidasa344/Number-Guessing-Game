# Guess the Number Game 🎯

## Overview

Guess the Number is a simple Python command-line game where the computer randomly selects a number between **1 and 200**, and the player has **6 attempts** to guess it correctly.

The game provides hints after each guess, telling the player whether the guessed number is too high or too low.

---

## Features

- Random number generation between 1 and 200
- User-friendly command-line interface
- Maximum of 6 guessing attempts
- Hints for incorrect guesses
- Input validation
- Personalized messages using the player's name

---

## Technologies Used

- Python 3
- Random Module
- Time Module
